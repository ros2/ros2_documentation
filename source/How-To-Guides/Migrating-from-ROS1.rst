
Migrating from ROS 1 to ROS 2
=============================

These guides show how to convert existing ROS 1 packages to ROS 2. Some automatic conversion tools exist online (e.g. `Magical ROS2 Conversion Tool <https://github.com/DLu/roscompile/tree/main/magical_ros2_conversion_tool>`_) but are not exhaustive.
If you are stuck, use the tutorials (:doc:`Developing a ROS 2 package<Developing-a-ROS-2-Package>`) as a reference.
